# Orion — Autonomous AI Trading Agent

Multi-asset trading system using LightGBM regime classification for dynamic position sizing.

## Status
V20.9 — Multi-Asset Paper Trading (ETH, BTC, SOL)

## Core Principle
Orion does not predict the market. It selects when the market deserves to be played.

## Tech Stack
- Model: LightGBM (frozen V20.6.1)
- Data: OKX via CCXT (4H candles)
- Infra: AWS EC2 + systemd
- Alerts: Telegram Bot
- Language: Python 3.12
